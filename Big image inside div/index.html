About the sizing and positioning of an img

For object-fit to work, the image itself needs a width and height. In the OP's CSS, the images do not have width and/or height set, thus object-fit cannot work. The clue: width and height need NOT be the dimensions of the image itself! Think of it as if it were a div: If you want a div to fill its container, you will set:

width: 100%; height: 100%;

In the examples above, both images have object-fit: cover and object-position: center;, and you could sp end the whole fucking day wondering why the hell this is not working. The clue is only the first has a width and a height defined (in this case set to 100%).
